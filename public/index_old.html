My CV

Markdown (original) Json New HTML ODT PDF

  • Why are you using those formats

    The original file is written in markdown, all other files are generated using that markdown file (even json). For Html and Odt I use pandoc, for json I built my own markdown parser using nodejs (the same server it's powering this page). You can read more about this in my bitbucket repo

  • Why is so special this Json format

    Json is a compact machine human-readable easy-to-use format, and it is generated directly from a markdown file using nodejs. You can read more about this in my bitbucket repo
